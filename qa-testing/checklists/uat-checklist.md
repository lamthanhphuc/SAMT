# UAT Checklist (Product Owner)

## Business Flow Validation

- [ ] User can sign in and reach correct dashboard for their role.
- [ ] Core business operations complete without manual workaround.
- [ ] End-to-end journey from setup to output/report works as expected.
- [ ] Permission boundaries match business expectations.

## UI Usability

- [ ] Navigation is intuitive for Admin/Lecturer/Student personas.
- [ ] Forms and tables are easy to understand.
- [ ] Error messages are clear and actionable.
- [ ] Responsive layout is usable on standard laptop resolution.

## Performance Responsiveness

- [ ] Initial page load feels responsive.
- [ ] Search/filter/table interactions respond within acceptable time.
- [ ] Report generation feedback is clear while waiting.
- [ ] No blocking UI freeze during normal workload.

## Acceptance Sign-Off

- [ ] Product Owner signs off business readiness.
- [ ] Open defects are triaged with release decision.
- [ ] Go-live recommendation documented.
